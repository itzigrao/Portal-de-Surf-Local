# Portal de Surf Local - Garopaba

Projeto front-end desenvolvido para a avaliação da Unidade Curricular de Programação para Internet, do curso de CST em Sistemas para Internet no IFSC - Campus Garopaba.

## Sobre o Projeto

O site é um portal focado no turismo e lazer da região de Garopaba, criado com base nos requisitos do Tema 1 da atividade prática. 
A aplicação funciona como um guia de surf e inclui um formulário interativo para inscrições em campeonatos ou escolas de surf locais. 
A estrutura visual inicial teve como ponto de partida o "Template 1: Clássico" disponibilizado pela disciplina.

### Tecnologias Utilizadas

O desenvolvimento atendeu ao requisito de utilizar apenas tecnologias nativas da web, sem frameworks:

* **HTML5:** Estruturação do conteúdo em no máximo 4 seções principais, garantindo a semântica adequada.
* **CSS3:** Layout construído majoritariamente com Flexbox, utilizando o padrão de Cards para organização visual e estilização baseada em classes e IDs.
* **JavaScript:** Lógica de manipulação do DOM para criar interatividade e realizar a validação básica dos campos do formulário.
